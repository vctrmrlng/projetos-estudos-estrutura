# PRINCIPAIS COMANDOS DO DJANGO
1. pip install django -> comando para instalar o django
2. django-admin startproject nome_do_projeto . -> Cria um novo projeto django
3. python manage.py runserver -> roda o servidor
4. CRTL + C -> Fecha o servidor
5. python manage.py migrate -> Aplica as migrações pendentes
6. python manage.py createsuperuser -> Criação do super usuário
7. python manage.py startapp sistema -> Criação de um app 

# PRINCIPAIS COMANDOS DO TERMINAL
1. cls -> Limpa o terminal
2. cd .. -> volta uma pasta
